las solicitudes.

```ts showLineNumbers /appDir: true/
curl -v -X PUT
    --url 'https://api.lumarketo.cl/public/product'
    --header 'Authorization: Bearer THwi13of2EHnPsSdfYmrB7Q17P6soi'
    --data '{ "sku": 50, "price": 2560 "stocks": [{ "subsidiaryId": "XXXXXXXXXXXXXXXXXXX", "qty": 6000 }]}'
```
