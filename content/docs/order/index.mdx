## Obtener orden 

**URL**: `/order/`

**Método**: `GET`

**Autenticación requerida**: Si

**Parámetros de consulta**: `orderNumber=[number]` 

**Parámetros**

```json
{}
```

**Ejemplo.**

```ts showLineNumbers /appDir: true/
curl -v -X GET
    --url 'https://api.lumarketo.cl/public/order?orderNumber=15'
    --header 'Authorization: Bearer THwi13of2EHnPsSdfYmrB7Q17P6soi'
```

#### Error

**Condición** : Sí el token de acceso es inválido o ya ha realizado más de una vez la solicitud dentro del tiempo límite.

**Código** : `403 FORBIDDEN`

**Ejemplo.**

```json
{
    "message": "User is not authorized to access this resource with an explicit deny"
}
```

## Obtener ordenes por fecha

Debe incluir un token de acceso en el encabezado de **'authorization'** en todas las solicitudes.
